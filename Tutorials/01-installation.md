## install

```sh
$ npm install webpack --save-dev
$ npm i -D webpack

```

## package.json

```js
"scripts": {
    "start": "webpack --config webpack.config.js",
    "pro": "webpack --config webpack.pro.config.js",
    "dev": "webpack --config webpack.dev.config.js"
}

``` 
## run

```sh
$ npm start
$ npm run pro
$ npm run dev

```
